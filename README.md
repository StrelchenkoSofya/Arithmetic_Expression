Третья лабораторная работа в рамках курса "Алгоритмы и структуры данных"
============

![C++](https://img.shields.io/badge/C++-17-00599C?logo=cplusplus&logoColor=white)
![CMake](https://img.shields.io/badge/CMake-3.5-064F8C?logo=cmake&logoColor=white)

Описание
---------------
Учебный проект был создан в рамках курса "Алгоритмы и структуры данных" и представляет из себя программу, принимающую на вход строку, состоящую из
нескольких подстрок, отделенных друг от друга точкой с запятой, каждая из которых присваивает переменной значение некоторого арифметического выражения.


Ключевые слова
---------------
Деревья, авл-деревья, итераторы, хэш-таблицы, конечные автоматы, постфиксная и инфиксная формы записи, стек, очередь, сэт, битфилд, арифметические выражения,
наследование, посетители, шаблонные классы.
